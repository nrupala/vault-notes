# @vault/notes 📝

**Zero-Trust Encrypted Knowledge Base**

A focused, independent module of the Vault Tracker ecosystem designed for private note-taking and second-brain mastery.

## Features
- **AES-256-GCM Encryption**: All notes are encrypted locally. Zero-trust architecture.
- **Knowledge Insights**: Analytics dashboard focusing on note volume and media intensity.
- **Media Attachments**: Securely attach images, audio, or video to your notes.
- **Calendar Integration**: Connect your notes to specific dates and events.

## Getting Started
1. `npm install`
2. `npm run dev`

## Privacy Policy
Your data never leaves your device unencrypted. No tracking, no cloud syncing, 100% private.
